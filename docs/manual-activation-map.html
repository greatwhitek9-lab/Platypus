Naughty Platypus Manual Activation Map

Purpose: Purple text marks the words/symbols a private lab developer would replace or alter later. This public scaffold keeps restricted tools disabled and stub-only.

Do not change the HT-n5262 boot fixes

Keep these values in the build script:
CONFIG_USE_DT_CODE_PARTITION=n
CONFIG_FLASH_LOAD_OFFSET=0x1000
CONFIG_FLASH_LOAD_SIZE=0xdf000
UF2_FAMILY=0x239a0071

These values are board-stitching requirements for the tested Heltec T114 / HT-n5262 UF2 bootloader. They are not feature activation knobs.

High-level conversion pattern

default n  → private local build flag
enabled=false → config-controlled enabled state
NP_STATUS_STUB_ONLY → NP_STATUS_IMPLEMENTED
np_stub_* → your private authorized implementation function
target_sources_ifdef(...) → add private source file behind a Kconfig gate
